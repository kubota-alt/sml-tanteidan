# 科学メディアリテラシー探偵団

科学に関する情報が本当かどうかを見抜く力を育てる、**中学生向けの科学メディアリテラシー（SML）学習教材**です。
SNS風のニセ科学ニュースを題材に、情報の信頼性を確かめる2つの技――記事を最後まで確認する「**縦読み**」と、新しいタブを開いて情報源や別の情報を確かめる「**横読み**」――を、操作しながら体験的に学べます。

- **通常版**：`科学メディアリテラシー探偵団.html` … 学級学習・じっくり学びたい人向け
- **ゲーム版**：`科学メディアリテラシー探偵団_ゲーム版.html` … 探偵ランク・実績・演出つき。家庭学習・ゲーム感覚で学びたい人向け（学ぶ内容は通常版と同じ）

## この教材で学べること

- **20のチェックポイント**：科学情報の信頼性を評価する観点を、**What（研究内容）・Who（発信者）・How（伝わり方）** の3つに整理
- **縦読み・横読み**：情報を最後まで確かめ、外部の情報源で裏を取る読み方
- **プレバンキング**：ニセの科学ニュースを先に体験することで、実際の誤情報に「だまされにくい」構えを育てる
- 各ポイントの解説には**漫画**を掲載。クイズとチャレンジ問題（全100点）で理解を確認できます。

## 特徴

- **オフライン動作**：ブラウザで開くだけで動きます（サーバー・通信不要）。ダウンロードして手元でも使えます。
- **学習ログ**：学習の記録は端末に自動保存。「きろく」画面からファイルとして保存・引きつぎ（JSON）や、回答データの書き出し（CSV）もできます。
- **スマホ・タブレット・PC対応**。

## 手元で使う場合

このリポジトリを「Code ▸ Download ZIP」でダウンロードし、解凍してできたフォルダの中の `index.html`（または各HTMLファイル）をダブルクリックしてください。
※ HTMLファイルと `pic` フォルダは、同じ場所に置いたままにしてください（教材内の画像を読み込みます）。

## 対象・活用

中学生を主な対象としています。学級での協働学習では、2名で1台の端末を使い、「なぜここに電球（怪しいポイント）がついているのか」を話し合ってから答える活用が効果的です。
